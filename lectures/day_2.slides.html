Function parts¶

def say_hello_to(name):
    greeting = "Hello " + name 
    return greeting
  • def: keyword (telling to Python that we are about to make a function now)
  • say_hello_to: function name, followed by parentheses (without spaces!)
  • name: function parameter
  • return: the stuff function returns to the place where we call it
  • say_hello_to(name): calling a function
In [5]:
text = "world in peace!"
output = say_hello_to(text)
print(output)

Hello world in peace!

Multiple arguments¶

In [8]:
def smaller(a, b):
    min_num = a
    if b < a:
        min_num = b
    return min_num
In [9]:

print(small_number)

2
In [10]:
